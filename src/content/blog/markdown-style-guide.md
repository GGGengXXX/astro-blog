这篇是写作样式备忘，方便之后复制。

## 小标题

用二级标题拆段落，页面会更适合扫读。

## 列表

- 短句
- 单一意思
- 不要把整篇文章都写成列表

## 引用

> 有时候，先记录比先理解更重要。

## 代码

```js
const note = {
  type: 'diary',
  status: 'draft',
};
```

## 图片

之后可以把自己的截图、照片或手稿扫描件放到 `src/assets/`，再在文章里引用。
